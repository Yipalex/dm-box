# 微博私信备份助手

把你和某个联系人（或全部联系人）的微博私信聊天记录**备份到自己电脑上**，支持**增量备份**、媒体文件下载、全文搜索和漂亮的 HTML 导出。

- **只读**：不调用任何写接口（不发私信、不标已读），账号行为严格限制在"浏览历史"语义内
- **保守限速**：每页请求间隔 5–10 秒随机抖动、每 10 页长休 45–90 秒、遇限流指数退避、每日请求上限 400 次——宁慢勿快，把封号风险降到最低
- **数据不出本机**：插件只和 `127.0.0.1` 上的本地服务通信，所有数据存在你电脑的 `data/` 目录里

## 系统组成

```
┌─────────────────────────┐      POST JSON       ┌──────────────────────────┐
│ Chrome 插件              │  ──────────────────▶ │ 本地服务 (FastAPI)        │
│ 在微博私信页注入备份面板    │   127.0.0.1:8765    │ SQLite + 媒体下载 + 导出   │
│ · 主动翻页拉取(保守限速)   │                      │ · 管理界面(联系人/搜索)    │
│ · 被动捕获(你翻看时顺路存) │                      │ · HTML 导出               │
└─────────────────────────┘                      └──────────────────────────┘
```

- **全量备份**：从最新消息向更早翻页（`mid` 游标），直到拉完
- **增量备份**：从最新往回拉，碰到数据库里已有的 `mid` 就停——通常一两页就完成
- **被动捕获**：你自己正常翻看聊天时，插件把微博页面"顺路"返回的数据存下来，零额外请求

## 快速开始

### 第 1 步：启动本地服务

需要 Python 3.10+（macOS 自带）。**双击项目里的 `start.command`**（Windows 用 `start.bat`）。
首次运行会自动安装依赖，然后浏览器会打开管理界面 <http://127.0.0.1:8765>。

> 命令行方式：`./.venv/bin/uvicorn server.app:app --host 127.0.0.1 --port 8765`

### 第 2 步：安装 Chrome 插件（开发者模式，2 分钟）

1. 打开 Chrome，地址栏输入 `chrome://extensions` 回车
2. 打开右上角的 **「开发者模式」** 开关
3. 点击左上角 **「加载已解压的扩展程序」**
4. 选择本项目里的 **`extension` 文件夹**
5. 完成！工具栏会出现橙色聊天气泡图标

### 第 3 步：开始备份

1. 登录微博后打开私信页：<https://api.weibo.com/chat>（点插件图标里的按钮也行）
2. 页面右下角会出现 **「私信备份助手」面板**，绿点表示已连上本地服务
3. 点 **「加载联系人」** → 在下拉框选择要备份的人
4. 第一次点 **「全量备份」**，之后日常点 **「增量备份」**（或选「全部联系人(增量)」）
5. 备份过程中**保持该标签页开着**；进度和日志都在面板里
6. 随时打开 <http://127.0.0.1:8765> 查看、搜索、导出 HTML

### 增量备份的日常用法

- 手动：隔几天打开私信页点一次「增量备份」，几秒到几十秒完成
- 自动：勾选面板里的「打开本页超过 24h 未备份时自动增量」，以后只要打开私信页就会自动补齐
- 顺路：平时正常翻看聊天记录，看到的内容会被被动捕获自动入库（零额外请求）

## 速率与风控策略（为什么慢）

| 策略 | 默认值 |
|---|---|
| 每页（200 条）请求间隔 | 5–10 秒随机 |
| 每 10 页（约 2000 条）长休 | 45–90 秒 |
| 多联系人之间休息 | 30–60 秒 |
| 每日主动请求上限 | 400 次（约 8 万条消息） |
| 遇 HTTP 414/418/432 或"访问频次过高" | 退避 2→4→8…分钟，连续 3 次即停止当天 |

一万条历史记录大约需要 8–15 分钟拉完；几十万条建议分几天跑（每天会自动在上限处停下，进度已保存，第二天接着增量即可）。**不要修改参数求快**——微博风控对高频规律请求敏感，慢就是稳。

> ⚠️ **风险告知**：本工具只读、限速保守，封号风险低但非零（更可能出现的是临时验证码/限流）。如果备份中出现验证码或面板提示连续限流，请停止当天操作。请仅用于备份你自己账号的私信。

## 跨平台支持（macOS / Windows）

核心代码（Python + FastAPI + SQLite）完全跨平台，Chrome 插件两边一样。差异只在启动脚本和两个可选的系统能力：

| 能力 | macOS | Windows |
|---|---|---|
| 启动本地服务 | 双击 `start.command` | 双击 `start.bat` |
| 防止运行时休眠 | 已内置 `caffeinate`（自动） | 控制面板→电源选项设为「从不睡眠」，或运行时用 `powercfg` |
| 语音 AMR→M4A 转码 | 自带 `afconvert`，开箱即用 | 需自行安装 [ffmpeg](https://www.gyan.dev/ffmpeg/builds/) 并加入 PATH；不装则语音存为 `.amr`（仍可下载，只是浏览器里点不开） |
| 开机自启后台常驻 | `./scripts/install_autostart.sh`（launchd） | 把 `start.bat` 的快捷方式放进「启动」文件夹（Win+R 输入 `shell:startup`） |

> Windows 用户：先装 [Python 3.10+](https://www.python.org/downloads/)（安装时勾选 "Add Python to PATH"），然后双击 `start.bat` 即可，其余步骤与 macOS 完全一致。

### 常驻运行（macOS）

- `start.command` 已内置 `caffeinate`：服务运行期间系统不会自动休眠（屏幕可以关，合盖仍会睡）
- 想让服务**开机自启、后台常驻**：终端运行一次 `./scripts/install_autostart.sh`，以后不用再双击启动；卸载用 `uninstall_autostart.sh`

> 注意（两个平台都适用）：**主动备份时浏览器标签页要保持在前台**。Chrome 会对后台标签限流定时器，备份会变得很慢；媒体下载由本地服务负责，不受此影响。

## 数据存放与格式

```
data/
├── weibo_dm.db        # SQLite 主存储（消息 + 原始 JSON + 联系人 + 同步日志）
├── media/             # 图片/视频/语音，按 SHA256 去重落盘
└── exports/           # 导出的 HTML 阅读视图
```

- 每条消息以 `mid` 为主键去重，**原始 JSON 整条入库**，未来可重建任意导出格式
- 媒体文件下载自带 `Referer: https://weibo.com`（绕过 sinaimg 防盗链），按内容哈希去重
- 想看原始数据：任何 SQLite 工具（如 [DB Browser for SQLite](https://sqlitebrowser.org)）打开 `weibo_dm.db` 即可

## 常见问题

- **面板红点 / 提示未连接本地服务**：先双击 `start.command` 启动服务
- **加载联系人失败**：确认已登录微博且当前页是 `api.weibo.com/chat`；微博接口偶尔变更，可开 F12 看实际请求
- **媒体没下载下来**：下载是后台慢速队列（每 2–4 秒一个），稍等；失败会自动重试 3 次
- **换电脑/迁移**：整个项目文件夹（连同 `data/`）拷走即可
- **接口失效**：微博的 `source=209678993` 等参数可能变更，见 `extension/content.js` 顶部常量

## 项目结构

```
extension/            # Chrome 插件 (Manifest V3)
├── manifest.json
├── content.js        # 备份面板 + 限速引擎 + 被动捕获接收
├── injected.js       # MAIN world：捕获页面自身的 conversation.json 响应
├── background.js     # 转发请求到本地服务
└── popup.html/js     # 工具栏弹窗
server/               # 本地服务 (FastAPI)
├── app.py            # API + 静态站点
├── db.py             # SQLite 存储层（mid 去重 / 原始 JSON / 媒体队列）
├── media_worker.py   # 媒体下载线程（限速 + 防盗链 Referer + SHA256 去重）
├── export.py         # HTML 导出
├── static/           # 管理界面
└── templates/        # 导出模板
scripts/
├── gen_icons.py      # 重新生成插件图标
└── pack_extension.sh # 打包插件 zip
```
